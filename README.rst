Hydrogen API
========

Hydrogen API allows you to access inside info on hydrogen bot.

RateLimit
---------

We currently allows 60 requests per 60 minutes

Features
--------

- Authorize
- Prefix
- Botinfo
- Guildinfo
- Userinfo
